# log_agent_project
#1. 综述  

一款轻量级的 C 语言日志采集代理，专注 Linux 平台日志实时采集与处理，目前测试过 Linux Ubuntu 平台  

#2. 如何使用  

可参考log_agent.c  

#3. 功能概要  

日志采集基于 inotify 实现日志文件变更实时监控；  

日志采集支持 ERROR 关键字精准过滤，仅捕获目标日志；  

日志采集支持批量刷盘机制（阈值 1024 行），平衡性能与实时性；  

日志采集支持多文件并发处理，适配生产级多日志场景；  

日志采集基于 mmap 实现高效文件读取，避免内存溢出；  

日志采集适配 Linux，支持 /var/log 目录日志采集；  

经压测：  

轻负载场景下 CPU 峰值≤30.6%、内存占用稳定≤8MB；  

10 文件并发采集场景下 CPU 峰值≤28.5%、磁盘写入速度≤0.8MB/s；  

极限场景下支持 10 万行日志无延迟采集，CPU 峰值≤62.3%、处理 TPS≥5000 行 / 秒
